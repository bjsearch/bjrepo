# 토스인슈어런스 보험 가이드라인 (참고 자료)

> 출처: 토스인슈어런스 보험 상담 가이드 PDF(사용자 업로드, 15p)를 정리한 참고 문서입니다.
> 이 리포트 생성기의 권장 보장금액 체크리스트(`rules_default.json`)와 인사이트 생성 로직이
> 참고하는 기준 자료로 사용합니다. 실제 상품 조건은 반드시 약관 원문으로 확인하세요.

## 보험 점검 4가지 기준

토스인슈어런스가 제시하는 보험 점검 체크리스트는 아래 4가지 기준으로 구성됩니다.

1. **범위(보장범위)** — 진단명이 얼마나 넓게 포함되는가
2. **크기(보장크기)** — 실제 치료비에 상응하는 금액인가
3. **기간(보장기간)** — 라이프사이클에 맞추어져 있는가 (갱신형 vs 비갱신형)
4. **보험료** — 과하지 않고 적정한가 (위험보험료 vs 적립보험료)

이 리포트의 "영역별 보장 진단" 섹션은 주로 1번(범위)·2번(크기) 기준을, "핵심 진단 및 제언"
섹션은 3번(기간)·4번(보험료) 기준을 반영하도록 설계되어 있습니다.

## 1. 보장범위 — 진단명 포함 관계

같은 "OO질환 진단비"라는 이름이어도 상품마다 포함 범위가 다릅니다. 넓은 범위를 포함하는
담보일수록 유리합니다.

### 암

| 분류 | 예시 |
|---|---|
| 일반암 | 폐암, 위암, 간암, 대장암 등 (소액암 제외 전체) |
| 유사암(소액암) | 갑상선암, 기타피부암, 경계성종양, 제자리암 |
| 특정소액암 | 전립선암, 자궁암, 유방암, 비침습방광암, 대장점막내암 |

* 보험사에 따라 "소액암"과 "유사암" 용어를 혼용합니다.
* 특정소액암이 일반암에 포함되는 상품이 고객에게 유리합니다.

### 뇌혈관질환 (범위: 뇌혈관질환 ⊇ 뇌졸중 ⊇ {뇌경색, 뇌출혈})

| 분류 | ICD 코드 | 뇌혈관질환 | 뇌졸중 | 뇌출혈 |
|---|---|:---:|:---:|:---:|
| 거미막하출혈 | I60 | O | O | O |
| 뇌내출혈 | I61 | O | O | O |
| 기타 비외상성 두개내출혈 | I62 | O | O | O |
| 뇌경색증 | I63 | O | O | - |
| 출혈/경색증 미명시 뇌졸중 | I64 | O | O | - |
| 뇌경색증 유발 안 한 뇌전동맥 폐쇄·협착 | I65 | O | O | - |
| 뇌경색증 유발 안 한 대뇌동맥 폐쇄·협착 | I66 | O | O | - |
| 기타 뇌혈관질환 | I67 | O | - | - |
| 달리 분류된 질환에서의 뇌혈관장애 | I68 | O | - | - |
| 뇌혈관질환의 후유증 | I69 | O | - | - |

"뇌혈관질환 진단비"가 뇌졸중·뇌출혈을 모두 포함하는 가장 넓은 범위의 담보입니다.

### 심장질환 (범위: 심장질환 ⊇ 허혈성심장질환 ⊇ {급성심근경색, 협심증})

| 분류 | ICD 코드 | 심장질환 | 허혈성 | 심근경색 |
|---|---|:---:|:---:|:---:|
| 협심증 | I20 | O | O | - |
| 급성심근경색증 | I21 | O | O | O |
| 후속 심근경색증 | I22 | O | O | O |
| 급성심근경색증 후 특정 합병증 | I23 | O | O | O |
| 기타 급성 허혈성심장질환 | I24 | O | O | - |
| 만성 허혈심장병 | I25 | O | O | - |
| 급성 심장막염 | I30 | O | - | - |
| 심장막의 기타질환 | I31 | O | - | - |
| 급성 및 아급성 심내막염 | I33 | O | - | - |
| 심부전 | I50 | O | - | - |

허혈성심장질환은 "가성비적으로 제일 적정한 보장"으로 평가됩니다(급성심근경색·협심증 포함).

## 2. 보장크기 — 실제 치료비 벤치마크

(출처: 질병관리청 등 통계, 2020년대 중반 기준 추정치 — 실제 비용은 병원·시기에 따라 변동)

| 항목 | 실제 비용 | 비고 |
|---|---|---|
| 암 수술비 | 약 200만원 | 산정특례(본인부담 경감) 적용 시 |
| 중입자방사선 치료 | 4,000만~7,000만원 | 산정특례 최대 5년 적용 후에도 고액 |
| 표적항암약물 치료 | 7,000만원 이상 | 신의료기술, 비급여 비중 높음 |
| 급여 뇌혈관스텐트 시술 | 약 180만원 | 뇌혈관질환 산정특례는 단 1개월만 적용 |
| 심장 스텐트삽입술 | 약 240만원 | 심장질환 산정특례도 단 1개월만 적용 |

**패러다임 변화**: 의료기술 발전으로 생존율이 오르면서, "1회성 진단비" 중심 설계보다
"치료 시마다 반복 지급되는 치료비" 중심 설계가 중요해지고 있습니다. 특히 암(표적항암·
중입자 등 고액 비급여 반복치료)과 뇌·심장질환(스텐트삽입·혈전제거 등 반복 시술)이 대표적입니다.

### 보장 우선순위 (발생확률 기준)

1. 암 (일반암, 소액암)
2. 뇌혈관 (뇌혈관질환, 뇌졸중, 뇌출혈)
3. 심장 (심장질환, 허혈성질환, 심근경색, 부정맥)
4. 수술 (질병/상해수술 1~5종, N대수술, 뇌심수술)
5. 항암/장해 (항암 방사선·약물, 골절, 상해/질병 후유장해)

## 3. 보장기간 — 라이프사이클별 전략

| 시기 | 핵심 | 권장 구조 |
|---|---|---|
| 자산형성기 (20~50대) | 재산 방어 — 목돈 소요 방지 | 비갱신형 기본, 집중 보장구간엔 갱신형 보조 활용 가능 |
| 은퇴·노후기 (60대 이후) | 지출 통제, 안정적 현금흐름 | 비갱신형 100세만기 + 납입면제, 연금 확보 필수 |

"가장 완벽한 설계는 돈을 버는 시기에 보장을 완성하고, 돈을 쓰는 시기에 혜택을 누리는 것"

## 4. 보험료 — 구조와 납입기간

보험료는 위험보험료(다른 가입자 보험금 재원, 약 1/3) + 운용보험료(계약 유지·관리 비용,
약 1/3) + 적립보험료(저축성, 예시 기준 약 1/3)로 구성됩니다. 적립보험료는 필요 시 인출이
자유롭지 않은 경우가 많아, 순수 보장만 원한다면 적립보험료 비중이 낮은(또는 없는) 상품이
저렴합니다.

| 납입기간 | 장점 | 단점 |
|---|---|---|
| 20년납 | 총 납입보험료가 30년납보다 적음 | 월 보험료가 30년납보다 높음 |
| 30년납 | 월 보험료가 20년납보다 낮음 | 총 납입보험료가 20년납보다 많음 |

연령대가 높거나 납입면제 기능을 크게 활용하고 싶다면 30년납, 연령대가 낮고 건강할수록
20년납이 합리적입니다.

## 피해야 할 보험 구조 (주의 신호)

| No. | 보종/구조 | 문제점 |
|---|---|---|
| 1 | 종신보험+보장특약 위주 | 사망보장·보장 모두 낮음 |
| 2 | CI보험 | 보장 조건(약관상 요건)이 매우 까다로움 |
| 3 | GI보험 | 보장 범위가 좁고 최초 1회한 지급 |
| 4 | 갱신형 위주 구성 | 연령 상승에 따른 보험료 상승 우려 |
| 5 | 적립보험료 포함형 | 보험료 낭비 · 불필요한 비용 |
| 6 | 만기가 짧은 보험 (70·80세 만기) | 정작 발병률이 높아지는 고령기에 보장 공백 |

## 이 리포트 생성기에서의 활용 방식

- `guarantee_report/guideline_notes.json`: 위 "보장크기" 표의 실제 치료비 벤치마크를
  기계가 읽을 수 있는 형태로 정리해, 영역별 보장 진단 표의 해당 항목에 참고 문구로
  덧붙입니다 (`rules.py`가 로드하여 `EvaluatedRow.note`에 채움).
- `rules_default.json`의 카테고리 계층 구조(뇌혈관질환/뇌졸중/뇌출혈, 심장질환/허혈성심장/
  급성심근경색)는 위 "보장범위" 표의 포함 관계를 그대로 반영합니다.
- "핵심 진단 및 제언" 섹션은 "피해야 할 보험 구조" 표를 참고해, 갱신형 비중이 높거나
  만기가 짧은(70·80세) 계약이 있으면 관련 인사이트를 생성합니다.
